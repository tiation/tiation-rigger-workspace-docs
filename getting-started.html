Getting Started

Prerequisites

  • Node.js (v18 or higher)
  • npm or yarn
  • Git
  • Docker (for containerized deployment)
  • MongoDB (for database)
  • Redis (for caching)

Quick Start

1. Clone the Repositories

git clone https://github.com/tiation/tiation-rigger-automation-server.git
git clone https://github.com/tiation/tiation-rigger-mobile-app.git
git clone https://github.com/tiation/tiation-rigger-connect-app.git
git clone https://github.com/tiation/tiation-rigger-jobs-app.git
git clone https://github.com/tiation/tiation-rigger-shared-libraries.git

2. Setup Shared Libraries

cd tiation-rigger-shared-libraries
npm install
npm run build
npm link

3. Setup Backend (Automation Server)

cd tiation-rigger-automation-server
npm install
npm link @tiation/rigger-shared
cp .env.example .env
# Edit .env with your configuration
npm run dev

4. Setup Frontend Applications

RiggerConnect App

cd tiation-rigger-connect-app
npm install
npm link @tiation/rigger-shared
npm run dev

RiggerJobs App

cd tiation-rigger-jobs-app
npm install
npm link @tiation/rigger-shared
npm run dev

Mobile App

cd tiation-rigger-mobile-app
npm install
npm link @tiation/rigger-shared
npx react-native run-ios  # or run-android

Environment Configuration

Backend (.env)

PORT=3000
NODE_ENV=development
MONGODB_URI=mongodb://localhost:27017/riggerconnect
REDIS_URL=redis://localhost:6379
JWT_SECRET=your-jwt-secret
API_BASE_URL=http://localhost:3000

Frontend (.env.local)

REACT_APP_API_URL=http://localhost:3000
REACT_APP_ENVIRONMENT=development

Docker Setup

Using Docker Compose

docker-compose up -d

Individual Services

# MongoDB
docker run -d -p 27017:27017 --name mongodb mongo:latest

# Redis
docker run -d -p 6379:6379 --name redis redis:latest

# Application
docker build -t rigger-automation-server .
docker run -p 3000:3000 rigger-automation-server

Development Workflow

  1. Always update shared libraries first
  2. Run tests before committing
  3. Use conventional commits
  4. Create feature branches
  5. Submit pull requests for code review

Testing

Unit Tests

npm test

Integration Tests

npm run test:integration

E2E Tests

npm run test:e2e

Deployment

Staging

npm run build
npm run deploy:staging

Production

npm run build:prod
npm run deploy:prod

Troubleshooting

Common Issues

  • Port already in use: Change the port in .env file
  • MongoDB connection failed: Ensure MongoDB is running
  • Module not found: Run npm install and check dependencies
  • Permission denied: Check file permissions and ownership
